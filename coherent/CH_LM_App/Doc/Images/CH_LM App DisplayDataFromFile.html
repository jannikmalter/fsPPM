CH_LM App DisplayDataFromFile.vi

This VI opens a data file reads the contents and displays them on the frontpanel.

Front Panel

Controls and Indicators

 Path

Filename of data to display.

Close

Click to close this window.

Print

Click to print this window.

Statistics Type

    		Serial

    Serial number of LabMax device

    		Mean

    Displays the average of the last N (Sample Size) samples.

    		Std. Dev.

    Displays the standard deviation of the last N (Sample Size) samples.

    		Min

    Displays the minimum value in the last N (Sample Size) samples.

    		Max

    Displays the maximum value in the last N (Sample Size) samples.

    		Counts

    Displays the number of measurements taken in the current data set. Counter resets when "Reset" button is pressed.

column

Data file

Name of selected data file

Date and Time

Date and Time of measured data.

Waveform Graph

Display of measured data.

Use the available controls to manipulate the display.
